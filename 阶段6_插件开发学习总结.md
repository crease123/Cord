# Yank Note 插件开发学习总结

## 一、插件系统概述

Yank Note 的所有功能都是通过插件系统实现的。内置了约 80+ 个插件，用户可以编写自定义插件扩展功能。

### 插件类型

| 类型 | 位置 | 使用场景 |
|------|------|----------|
| 内置插件 | `src/renderer/plugins/` | TypeScript，需要重新编译 |
| 外部插件 | `<用户主目录>/plugins/` | JavaScript，热加载即可使用 |

## 二、插件基本结构

### 最简单的插件

```javascript
// 外部插件（plugin-hello.js）
window.registerPlugin({
  name: 'plugin-hello',
  register: ctx => {
    // 插件逻辑
  }
})
```

### TypeScript 内置插件

```typescript
// 内置插件（xxx.ts）
import type { Plugin } from '@fe/context'

export default {
  name: 'plugin-name',
  register: ctx => {
    // 插件逻辑
    return {
      // 可选：导出 API 供其他插件使用
    }
  }
} as Plugin
```

## 三、Context (ctx) 核心模块

`ctx` 是插件开发的核心，包含以下模块：

### 3.1 UI 相关

| 模块 | 用途 | 示例 |
|------|------|------|
| `ctx.ui.useToast()` | 显示提示信息 | `ctx.ui.useToast().show('info', '消息')` |
| `ctx.ui.useModal()` | 弹出对话框 | `ctx.ui.useModal().alert({ title, content })` |
| `ctx.ui.useContextMenu()` | 右键菜单 | - |
| `ctx.ui.useQuickFilter()` | 快速过滤器 | - |

### 3.2 编辑器相关

| 模块 | 用途 | 示例 |
|------|------|------|
| `ctx.editor` | 编辑器操作 | `ctx.editor.getLineContent(line)` |
| `ctx.editor.getEditor()` | 获取 Monaco 实例 | `ctx.editor.getEditor().getSelection()` |
| `ctx.editor.whenEditorReady()` | 等待编辑器就绪 | - |

### 3.3 文档与视图

| 模块 | 用途 |
|------|------|
| `ctx.doc` | 文档操作（打开、保存、切换） |
| `ctx.view` | 预览视图操作 |
| `ctx.tree` | 文件树操作 |

### 3.4 系统功能

| 模块 | 用途 |
|------|------|
| `ctx.action` | Action 注册和管理 |
| `ctx.registerHook` | Hook 注册 |
| `ctx.statusBar` | 状态栏菜单 |
| `ctx.setting` | 设置管理 |
| `ctx.i18n` | 国际化 |
| `ctx.theme` | 主题和样式 |
| `ctx.markdown` | Markdown 渲染扩展 |
| `ctx.keybinding` | 快捷键常量 |

## 四、核心功能详解

### 4.1 注册状态栏菜单

```typescript
ctx.statusBar.tapMenus(menus => {
  menus['my-menu'] = {
    id: 'my-menu',
    position: 'left', // 'left' | 'right'
    title: '菜单标题',  // 字符串或 Vue 组件
    order: 100,        // 排序，越大越靠后
    list: [
      {
        id: 'item-1',
        type: 'normal',
        title: '菜单项',
        onClick: () => { /* 处理逻辑 */ }
      },
      { type: 'separator' }, // 分隔线
    ]
  }
})
```

### 4.2 注册 Action（可绑定快捷键）

```typescript
ctx.action.registerAction({
  name: 'plugin.my-plugin.action-name',
  description: '动作描述（显示在命令面板）',
  keys: [ctx.keybinding.Ctrl, ctx.keybinding.Shift, 'k'], // 快捷键
  forUser: true, // 是否在命令面板显示
  handler: (arg1, arg2) => {
    // 处理逻辑
  }
})

// 调用 Action
ctx.action.getActionHandler('plugin.my-plugin.action-name')(arg1, arg2)
```

### 4.3 注册 Hook

```typescript
// 注册
ctx.registerHook('HOOK_NAME', async (payload) => {
  // 处理逻辑
  return false // 返回 true 可中断后续 hook
})

// 触发自定义 Hook
ctx.triggerHook('MY_CUSTOM_HOOK', payload)
```

**常用内置 Hook：**

| Hook 名称 | 触发时机 |
|-----------|----------|
| `DOC_BEFORE_SWITCH` | 切换文档前 |
| `DOC_SWITCHED` | 切换文档后 |
| `VIEW_ELEMENT_CLICK` | 预览区点击 |
| `EDITOR_PASTE_IMAGE` | 粘贴图片 |
| `MARKDOWN_BEFORE_RENDER` | Markdown 渲染前 |

### 4.4 注册设置项

```typescript
ctx.setting.changeSchema(schema => {
  // 添加设置分组（可选）
  schema.groups.push({
    label: '我的插件',
    value: 'my-plugin'
  })

  // 添加设置项
  schema.properties['plugin.my-plugin.setting-key'] = {
    defaultValue: true,
    title: '设置标题',
    description: '设置描述',
    type: 'boolean', // 'boolean' | 'string' | 'number'
    format: 'checkbox', // 'checkbox' | 'textarea' 等
    group: 'my-plugin',
    required: true
  }
})

// 读取设置
const value = ctx.setting.getSetting('plugin.my-plugin.setting-key', defaultValue)
```

### 4.5 扩展 Markdown 渲染

```typescript
ctx.markdown.registerPlugin(md => {
  // md 是 markdown-it 实例
  md.use(somePlugin)
  
  // 自定义渲染规则
  md.renderer.rules.my_rule = (tokens, idx, options, env, self) => {
    return '<div>自定义渲染</div>'
  }
})
```

### 4.6 添加样式

```typescript
ctx.theme.addStyles(`
  .my-class {
    color: red;
  }
`)
```

## 六、外部插件快速开发

### 创建步骤

1. 右键托盘图标 → 打开主目录
2. 在 `plugins/` 文件夹创建 `plugin-xxx.js`
3. 编写插件代码
4. 右键托盘 → 开发 → 重新加载

### 简单示例：快速插入时间戳

```javascript
// plugins/plugin-insert-timestamp.js
window.registerPlugin({
  name: 'plugin-insert-timestamp',
  register: ctx => {
    ctx.action.registerAction({
      name: 'plugin.insert-timestamp.now',
      description: '插入当前时间戳',
      keys: [ctx.keybinding.Ctrl, ctx.keybinding.Shift, 't'],
      forUser: true,
      handler: () => {
        const timestamp = ctx.lib.dayjs().format('YYYY-MM-DD HH:mm:ss')
        ctx.editor.insert(timestamp)
      }
    })

    ctx.statusBar.tapMenus(menus => {
      menus['insert-timestamp'] = {
        id: 'insert-timestamp',
        position: 'left',
        title: '⏰',
        onClick: () => ctx.action.getActionHandler('plugin.insert-timestamp.now')()
      }
    })
  }
})
```
